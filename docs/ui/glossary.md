# UI glossary (translator source)

| Term | Definition |
|------|------------|
| Project | A client or site workspace; crawls, keywords, and reports belong to one project unless exported. |
| Job | One tracked run (crawl, audit, SERP batch, …); can fail, pause, or degrade; shown in the Job dock. |
| Snapshot | One captured SERP HTML + parsed rows at a moment in time. |
| Ranking | Position derived from a snapshot or manual/API import (`provenance` explains which). |
| Degraded | Run finished but some steps failed — partial data is saved. |
| Blocked | Site or directory refused automated access — not a bug; try manual or proxy. |
| Politeness preset | Named bundle of delays and concurrency — not “VPN speed”. |
| Location | One business NAP entity (golden record) used by Citations. |
