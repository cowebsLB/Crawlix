# Issue Taxonomy and Priority

## Canonical Categories
- technical_health
- indexability
- duplication
- internal_linking
- metadata
- performance
- content_quality
- crawl_efficiency

## Severity vs Priority
- Severity: technical impact level.
- Priority: urgency in project context.

## Priority Heuristics
Inputs include:
- severity score
- confidence score
- internal inbound importance proxy
- blast radius (affected page count)
- recurrence count

Output buckets:
- later
- soon
- now

No direct severity==priority mapping.
