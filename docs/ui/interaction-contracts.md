# Interaction Contracts

## Page Frame
All data-heavy pages use:
1. Command Bar
2. Results Workspace
3. Persistent Right Inspector

## Command Bar
- Primary action first.
- Secondary actions grouped.
- Quick filters + saved view controls always visible.

## Table Contract
- Predictable row selection.
- Stable sort/filter behavior.
- Context menus aligned with next actions.

## Inspector Contract
- Single-row selection drives context.
- Shows category, severity, priority, confidence, evidence, recommendation.
- Empty state is explicit and non-ambiguous.

## Job Center Contract
- Global panel + inline page status.
- Unified states: queued, running, blocked, failed, completed, cancelled.
